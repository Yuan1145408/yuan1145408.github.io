C++ 入门与语法概览

快速掌握现代 C++（含 C++11+）的基础、STL、编译与调试。

第二步：怎么做

  1. 准备编译器（g++ 或 MSVC），确保 g++ --version / cl 可用
  2. 写 main.cpp 的 Hello World 并成功运行
  3. 学习头文件与源文件分离，尝试 CMake 构建
  4. 使用断点与调试器查看变量与调用栈

基础语法

// g++
g++ -std=c++17 main.cpp -o app
./app

// MSVC
cl /EHsc /std:c++17 main.cpp && app.exe

项目实战（入门）

  1. JSON 解析：使用 nlohmann/json 读取并处理数据
  2. 并发下载：std::thread + 限流 + 合并结果
  3. 文件工具：std::filesystem 遍历并生成报告

下一步

  • CMake/Ninja 构建与单元测试
  • STL 容器与算法的进阶用法
  • 异常、安全与性能优化

语法速览

  • 范围 for：for (auto &x : v) { ... }
  • 结构化绑定：auto [k, v] = pair;
  • 字符串拼接：std::string s = a + b;
  • 格式化：std::format("{} {}", a, b) (C++20)

常见坑

  • 空悬引用/指针：确保对象生命周期与所有权清晰
  • 异常安全：使用 RAII，防止资源泄漏
  • 拷贝 vs 移动：优先使用移动构造与 std::move

练习题

  • 实现一个 LRU 缓存（map+list 或 unordered_map+list）
  • 用 std::thread 实现并发下载并合并结果
  • 用 std::filesystem 遍历目录，生成文件树 JSON

国内网络建议

  • 使用国内镜像安装依赖与编译工具
  • 遇到网络受限，优先使用本地“国内模式”与代理下载

参考资源

系统学习路径

  1. 语法与 STL：基础语法、移动语义、容器与算法、异常与 RAII
  2. 工程化：现代 CMake、单元测试（Catch2/GTest）、类型与格式化
  3. 并发与网络：线程与原语、异步模型、常用网络库与部署
