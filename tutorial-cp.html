竞赛环境

  • 本地环境：VS Code + C++/Python/Java；OJ 常用编译参数与快速模板。
  • 调试技巧：断点、输入重定向、快速评测脚本（Python）。
  • 常见 OJ：Codeforces/AtCoder/洛谷/牛客；如何注册与参赛。

题目路线

  1. 基础数据结构与算法模板打包。
  2. 专项专题：DP（状态与转移/区间/状态压缩）、图论（最短路/网络流）、数学（数论/组合）。
  3. 比赛实战：模拟赛与复盘（题目分类、时间分配、错误分析）。

模板库与训练方法

  • 个人模板库维护：按模块存放、添加说明与用例。

  • 团队协作：分工、题目分配、沟通与中台策略。
