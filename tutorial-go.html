Go 入门与语法速览

覆盖环境安装、go mod、并发与通道、标准库（net/http、json、os/fs）、测试与构建。强调“快速能跑，再优化”。

完整新手路径：第一步怎么做

  1. 安装 Go（推荐 1.21+），确认 go version 可用。
  2. 初始化项目：mkdir hello && cd hello && go mod init example.com/hello
  3. 创建 main.go 输出 Hello World 并运行：go run .
  4. 添加一个第三方包（如 golang.org/x/sync），熟悉依赖管理。
  5. 写一个小函数并配套测试，形成“写-测-跑”的节奏。

并发与通道

ch := make(chan int)
go func(){ ch <- 42 }()
v := <-ch
fmt.Println("got", v)

// WaitGroup + goroutine
var wg sync.WaitGroup
wg.Add(1)
go func(){ defer wg.Done(); fmt.Println("work") }()
wg.Wait()

构建与发布

  • 本地构建：go build -o app，运行 ./app。
  • 交叉编译：GOOS=linux GOARCH=amd64 go build（PowerShell 使用 $env:GOOS 设定）。
  • 模块与版本：使用 tag 进行版本化，配合 CI 发布二进制。

国内网络建议

  • 设置 GOPROXY 国内代理，如：https://goproxy.cn
  • 遇到拉取依赖失败，先检查代理与网络环境，或临时使用镜像。
