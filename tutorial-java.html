完整新手路径：第一步怎么做

  1. 安装 JDK（推荐 Adoptium Temurin 17 LTS），确认 java -version 可用。
  2. 安装 VS Code 或 IntelliJ IDEA，熟悉终端 Console。
  3. 新建 Main.java 写 Hello World，先跑起来。
  4. 尝试简单表达式 1+2，形成“修改-保存-运行”的节奏。
  5. 了解 Maven/Gradle 构建工具，后续用于依赖管理与打包。

  double s = 0; for(T x: list) s += x.doubleValue(); return s;
}

并发与线程池

import java.util.concurrent.*;
ExecutorService pool = Executors.newFixedThreadPool(4);
Future<Integer> f = pool.submit(() -> 1+2);
System.out.println(f.get());
pool.shutdown();

// CompletableFuture 组合
CompletableFuture<Integer> a = CompletableFuture.supplyAsync(() -> 1);
CompletableFuture<Integer> b = CompletableFuture.supplyAsync(() -> 2);
CompletableFuture<Integer> c = a.thenCombine(b, Integer::sum);
System.out.println(c.join());

注解与反射

import java.lang.annotation.*;
@Retention(RetentionPolicy.RUNTIME)
@Target(ElementType.TYPE)
@interface Model { String table(); }

@Model(table = "users")
class User { public String name; }

Class<?> cls = User.class;
Model m = cls.getAnnotation(Model.class);
System.out.println(m.table());
