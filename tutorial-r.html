R 语言入门（数据分析）

数据清洗、可视化与统计建模的最小路径。

安装

  • 安装 R 与 RStudio：CRAN、RStudio

数据框与可视化

# 安装包：install.packages("tidyverse")
library(tidyverse)

# 读取 CSV
df <- read_csv("data.csv")

# 过滤、分组、聚合
df2 <- df %>% filter(score > 60) %>% group_by(city) %>% summarise(avg = mean(score))

# 可视化
ggplot(df2, aes(x = city, y = avg)) + geom_col()

建模（示例）

model <- lm(mpg ~ cyl + hp, data = mtcars)
summary(model)
