React 入门（使用 Vite）

组件、状态、路由的最小闭环；从零创建到运行。

环境准备

  • 安装 Node.js（建议 LTS）：nodejs.org
  • 安装 pnpm 或 npm：npm i -g pnpm（可选）

创建项目

# 使用 Vite 创建 React 项目
npm create vite@latest my-react-app -- --template react
cd my-react-app
npm install
npm run dev

打开浏览器访问控制台显示的本地地址，看到页面即可。
