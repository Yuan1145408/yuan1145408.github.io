C# 上手指南（.NET）

基础语法、项目结构、异步与 LINQ、常见问题与实践。跟着本文从入门到能做项目。

给完全新手：第一步怎么做

  1. 安装 .NET SDK，运行 dotnet --info 查看版本。
  2. 新建控制台项目：dotnet new console -n HelloApp。
  3. 在 Program.cs 打印 Hello，并尝试 Task.Delay。
  4. 用 VS Code 或 VS 开启断点，观察变量变化。
  5. 理解 dotnet run 与 dotnet build 的区别。

安装与环境

  • .NET SDK：使用 dotnet 命令，新建项目 dotnet new console。
  • IDE：VS Code + C# 扩展 或 Visual Studio Community。

基础语法

// 记录与不可变
public record User(string Name, int Age);
// 模式匹配
static string Classify(object o) => o switch {
  int n when n > 0 => "positive",
  string s => $"str:{s}",
  _ => "unknown"
};
// 异步
static async Task<string> FetchAsync(string url){
  using var http = new HttpClient();
  return await http.GetStringAsync(url);
}

常见坑

  • 异步死锁：UI 环境避免 .Result/.Wait()，使用 await。
  • 可空用法：启用 nullable，合理使用 ? 与 !。
  • 资源释放：优先 using 与 IAsyncDisposable。

练习题（基础）

  1. 实现一个 CLI 下载器，支持断点续传与取消。
  2. 编写一个 LINQ 查询管道，对对象列表做聚合。
  3. 创建一个 ASP.NET Minimal API，返回 JSON 列表。

项目实战（入门）

  1. 命令行工具：读取文本统计词频，支持参数与帮助。
  2. Web API：最小 API 返回 /ping，用 HttpClient 调用。
  3. 文件处理：遍历目录、重命名、日志记录，支持回滚。

国内网络建议

  • NuGet 国内镜像：dotnet nuget add source 添加镜像源。
  • 遇到网络受限：优先使用本地“国内模式”与代理下载。
  • 部署：使用 dotnet publish -r win-x64 --self-contained 生成可执行文件分发。

系统学习路径

  1. 语言基础与面向对象：类型系统、泛型、LINQ、异步与内存模型。
  2. 工程与测试：项目结构、DI、xUnit/NUnit、日志与配置管理。
  3. Web 与持久化：ASP.NET Core、EF Core、容器化与部署流水线。

进阶项目

  1. 模块化后端：分层/DDD、Repository、统一异常与响应格式。
  2. 高并发 API：异步 I/O、连接池与缓存、指标与可观测性。
  3. 多租户 SaaS：租户隔离、计费与审核、后台与运维工具。
