HTML + CSS 上手指南

语义化、布局、响应式与组件化样式，以及常见坑。

HTML 语义化

  • 结构：<header> <main> <section> <footer>
  • 可访问性：合理使用 alt、标题层级与表单标签

CSS 布局

/* Flex */
.row{display:flex;gap:12px;align-items:center}
/* Grid */
.grid{display:grid;grid-template-columns:repeat(3,1fr);gap:12px}
/* 媒体查询 */
@media (max-width:600px){.grid{grid-template-columns:1fr}}

常见坑

  • 盒模型：使用 box-sizing: border-box; 更好地控制宽高
  • 优先级：少用 !important，提高选择器可维护性
  • 响应式：使用流式布局与媒体查询
