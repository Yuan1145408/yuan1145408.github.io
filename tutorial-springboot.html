Spring Boot 入门

使用 Spring Initializr 创建项目，编写 REST 控制器并联通数据。

创建项目

  • 访问 start.spring.io 生成项目（依赖：Spring Web）。
  • 解压后用 IDEA 打开，等待依赖下载完成。

控制器

// src/main/java/com/example/demo/HelloController.java
package com.example.demo;
import org.springframework.web.bind.annotation.*;

@RestController
@RequestMapping("/api")
public class HelloController {
  @GetMapping("/hello")
  public Object hello(){ return java.util.Map.of("msg","hello"); }
}

运行

// DemoApplication 的 main 方法运行，默认端口 8080
