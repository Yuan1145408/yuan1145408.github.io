PHP 上手指南

安装、基础语法、Composer、Web 路由与常见误区、练习题等。

安装与环境

  • 安装：使用官方包或安装 php + composer。
  • 内置服务：php -S localhost:8080。

基础语法

<?php
// 数组与关联数组
$arr = [1, 2, 3]; $map = ['a' => 1, 'b' => 2];
// 函数与类型提示
function add(int $a, int $b): int { return $a + $b; }
// 类与命名空间
namespace Demo; class User { public function __construct(public string $name){} }
// Composer 自动加载
// composer init && composer require psr/log

给完全新手：第一步怎么做

  1. 安装 PHP 和 Composer：确保 php -v、composer -V 正常。
  2. 创建项目文件夹，初始化 composer init。
  3. 新建 index.php 输出 Hello，启动内置服务器 php -S localhost:8080。
  4. 添加自动加载与一个简单路由，访问不同路径返回不同内容。

项目实战（入门）

  1. 表单处理：提交、校验、保存为 CSV。
  2. REST API：返回 JSON 列表与分页。
  3. 文件上传：保存到指定目录并记录日志。

常见坑

  • 全等比较：使用 ===，避免隐式类型转换。
  • 错误处理：启用错误显示与日志，使用异常。
  • 安全：输入过滤、参数化查询，避免 SQL 注入。

练习题

  1. 实现一个简单路由器，返回不同页面。
  2. 编写一个表单提交与校验流程。
  3. 实现一个文件上传与保存功能。

中间插图

进阶内容

  • 依赖管理与自动加载：PSR-4、Composer 脚本与发布。
  • Web 框架：路由、中间件与控制器（Slim/Laravel）。
  • 安全与合规：输入过滤、CSRF、预处理语句与密码哈希。

更多练习

  1. 实现一个分页列表与搜索接口，返回 JSON。
  2. 编写一个中间件：记录请求日志与耗时。
  3. 做一个上传服务：限制文件类型与大小并记录。

系统化学习路径

  1. 语言与实战：语法、PSR 标准、错误与异常、测试。
  2. 工作流：Composer、自动加载、CI/CD、日志与配置。
  3. Web 与安全：路由与中间件、ORM、认证授权、输入输出安全。

进阶项目

  1. REST 服务：分页与过滤、统一错误、统计与限速。
  2. 上传与处理：分块上传、校验、图片/文档转换与预览。
  3. 多模块与钩子：权限系统、活动日志、配置中心与缓存。
