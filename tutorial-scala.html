Scala 入门

JVM 上的函数式编程；集合操作与并发模型。

示例

// build.sbt 中添加：scalaVersion := "2.13.12"
object Hello extends App {
  val nums = List(1,2,3,4).map(_ * 2).filter(_ > 4)
  println(nums) // List(6,8)
}
