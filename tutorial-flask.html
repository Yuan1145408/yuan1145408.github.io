Flask 入门

轻量、易上手，适合快速做原型或小型后端。

Hello, Flask

python -m pip install flask
# app.py
from flask import Flask, request, jsonify
app = Flask(__name__)

@app.get('/api/hello')
def hello():
    return jsonify({ 'msg': 'hello' })

@app.post('/api/echo')
def echo():
    return jsonify({ 'youSent': request.json })

if __name__ == '__main__':
    app.run(port=5050, debug=True)
